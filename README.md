**新拟态是一种图形样式，其原理是为界面的UI元素赋予真实感。**

## **介绍：**
#####  1、前端
 - 使用uni-app框架进行搭建，支持vue3、vue2
 - 使用拟态UI元素创作，页面布局flex
 - 基本功能已经全部实现，后期做初步升级优化
 - 
 
#####  2、后端
 - Thinkphp框架进行接口的搭建
 - 后台管理使用的vue2+elementUI进行的搭建（不在这里展示了）
 - 后台功能进行了简单的交互
